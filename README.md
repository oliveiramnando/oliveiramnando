## 🚀 What I'm Building

### [LeetNode]([https://github.com/oliveiramnando](https://leetnode-nine.vercel.app/))

A social platform for programmers to track LeetCode progress, share submissions, build streaks with friends, and compete through leaderboards.

* Built with **Next.js, TypeScript, Node.js, MongoDB, and Tailwind CSS**
* Implemented GitHub authentication and LeetCode account integration
* Designed submission feeds, friend systems, streak tracking, profiles, and leaderboards
* Deployed as a full-stack production application

### FinWiz

A financial management platform that helps users understand their finances, investments, and portfolio risk.

* Integrated **Plaid** for financial account and investment data
* Built a **Monte Carlo portfolio simulation service** using Python and FastAPI
* Calculated portfolio risk metrics including **VaR and CVaR**
* Implemented authentication, encrypted financial data storage, payments, and containerized services
